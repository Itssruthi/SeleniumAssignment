# SeleniumAssignment

<b>Account Creation in Fox.com.</b>

Two scenarios are covered:
1. Create Account with Email
2. Create Account FaceBook Sign In

Versions used for testing:
1. Selenium-Java : 3.8.1
2. Gecko Driver : geckodriver-v0.19.1-macos.tar
3. firefox : 57.0.4

<b>Note:</b> There could be many scenarios but this assignment covers the most common scenarios.
